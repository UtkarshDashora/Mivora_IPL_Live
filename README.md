# 🏏 Mivora: IPL Stadium Pulse

A premium, real-time stadium experience dashboard designed for IPL match attendees. Stay updated on live match schedules and stadium facility congestion with a stunning premium aesthetic.

## ✨ Features
- **Live Match Tracking:** Real-time feedback on live matches, upcoming clashes, and recently ended games.
- **Stadium Pulse:** Fluctuating, dynamic wait times for stadium entries (Gates) and food plazas (Fan Zones).
- **IPL Branding:** Immersive Navy and Gold aesthetic inspired by the energy of the Indian Premier League.
- **Dynamic Simulation:** Live data simulator for crowd management and stadium load balacing.

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abhishekmsm51w-star/mivora.git
   ```
2. **Navigate into the project directory:**
   ```bash
   cd mivora
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the fan dashboard:**
   ```bash
   npm run dev
   ```

## ☁️ Deployment
Optimized for Google Cloud Run on port `8080`.
```bash
gcloud run deploy mivora --source . --port 8080 --allow-unauthenticated --region us-central1
```
